<h2 id="一鍵借火自動借火">一鍵借火/自動借火</h2>
<p><img src="https://i.postimg.cc/NF4YFxTp/13.jpg" alt="借火功能截圖"></p>
<ul>
<li><strong>一鍵借火</strong>: 一鍵搜尋本地圖中擁有綠火的玩家並借取。</li>
<li><strong>自動借火</strong>:
<ol>
<li>啟動後，左上角會出現計時器（會自動判斷目前持有的是大綠火或小綠火）。</li>
<li>如果綠火效果消失，且此功能為開啟狀態，程式會自動再次搜尋地圖上是否有火源。若沒有，則會停止搜尋。</li>
<li>若先使用「一鍵借火」成功借到火，在效果結束後，「自動借火」功能仍會自動接續搜尋。</li>
</ol>
</li>
</ul>

<h2 id="自動戰鬥">自動戰鬥</h2>
<p><img src="https://i.postimg.cc/ryxF1cjG/1-1-4-22.jpg" alt="自動戰鬥功能截圖"></p>
<ul>
<li>會自動使用精靈四個技能中的第一招。</li>
<li>當技能沒有 PP 時會自動補充。</li>
<li>出戰順序與背包內的放入順序一致，建議戰鬥前先排好陣容。</li>
<li>進入巔峰對戰頁面時會自動關閉，因為此功能在巔峰中會卡住且不會自動出招。</li>
</ul>

<h2 id="每日助手">每日助手</h2>
<p><img src="https://i.postimg.cc/HnWRZ1c9/17.jpg" alt="每日助手截圖"></p>
<ul>
<li><strong>通用說明</strong>:
<ul>
<li>所有每日助手的精靈會自動使用第一個技能。</li>
<li>沒有 PP 時會自動補充。</li>
<li>出戰順序與背包內的放入順序一致，建議先排好陣容以避免失敗。</li>
</ul>
</li>
</ul>

<h3>每日戰場</h3>
<ul>
<li>自動從學習力戰場打到經驗值戰場 (只支援第六層)。</li>
<li>如果打完後又按開始，導致無法進入其他戰鬥，請按下停止按鈕。</li>
<li>若忘記關閉電池導致精靈升級，可使用緊急停止按鈕，然後重新開關電池。</li>
<li>若因任何情況中斷，可使用防呆系統從中斷的地方繼續。</li>
<li><code>輪次 = 完成數 + 1</code>。</li>
<li><code>第幾隻 = 現在要打第幾隻怪</code>。</li>
</ul>

<h3>精靈王試煉</h3>
<ul>
<li>選擇任意屬性的精靈王，並可選擇要挑戰的次數 (僅限困難版本)。</li>
</ul>

<h3>戰隊</h3>
<ul>
<li>一鍵完成5次貢獻。</li>
<li>可在任何地方打開任務面板。</li>
<li>戰隊道具中心 (此功能必須在戰隊地圖才能開啟，離開後記得開關電池)。</li>
</ul>

<h3>勇者之塔</h3>
<ul>
<li>開始掃蕩後，會自動倒數30分鐘並自動領取獎勵。</li>
<li>關掉助手重開也會繼續計時。</li>
<li>若關掉遊戲，計時器會重置，重開遊戲後請手動按「領取獎勵」。</li>
</ul>

<h3>新勇者之塔</h3>
<p><img src="https://i.postimg.cc/QxDNSPY2/1-1-4-33.jpg" alt="新勇者之塔截圖"></p>
<ul>
<li>會自動挑戰新勇者之塔的第30層共5次。</li>
<li>同樣提供防呆系統，可從中斷處繼續。</li>
</ul>

<h3>X戰隊密室</h3>
<ul>
<li>自動開啟副本並挑戰一次 (一天只能開啟三次，所以此按鈕只能按三次)。</li>
<li>可領取通關獎勵。</li>
</ul>

<h3>停止按鈕</h3>
<ul>
<li>如果每日助手發生任何情況導致無法正常戰鬥，請按下此按鈕。</li>
<li>如果按下後問題還在，請刷新遊戲。</li>
</ul>

<hr>

<h2 id="背包2.0">背包2.0</h2>
<p><img src="https://i.postimg.cc/9FBmkpvr/1-1-4-40.jpg" alt="背包2.0功能截圖"></p>
<h3>獲取背包精靈</h3>
<ul>
<li>點擊後，會將當前背包內的所有精靈，依照獲取時間由舊到新顯示在預覽格子中。</li>
</ul>

<h3>匯出背包精靈</h3>
<ul>
<li>匯出在預覽格子中的精靈組合。可以先調整格子順序再匯出。</li>
<li>檔案會儲存在 <code>/resources/Bag</code> 資料夾。</li>
</ul>

<h3>讀取精靈</h3>
<ul>
<li>讀取位於 <code>/resources/Bag</code> 資料夾中的 <code>.txt</code> 檔案，並將精靈顯示於預覽格子。</li>
</ul>

<h3>放入背包</h3>
<ul>
<li>讀取 <code>.txt</code> 檔案後，按下此按鈕會依照預覽格子中的順序將精靈放入背包。</li>
</ul>

<h3>開啟倉庫</h3>
<ul>
<li>開啟遊戲內的精靈倉庫介面。</li>
</ul>

<h3>放入倉庫</h3>
<ul>
<li>將背包內所有精靈全部放入倉庫。</li>
</ul>

<h3>快捷精靈</h3>
<ul>
<li>可將常用精靈放置於右側快捷欄。</li>
<li>點擊快捷欄的精靈可將其加入預覽格。按下「放入背包」後，會從倉庫中找出該精靈（若有多隻會抓取時間最新的）。</li>
<li>可按 "X" 從快捷欄刪除，或按 "+" 透過輸入精靈ID來新增。</li>
</ul>
